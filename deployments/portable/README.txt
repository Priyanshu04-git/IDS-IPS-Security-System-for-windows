# IDS/IPS Security System - Quick Reference

## 🚀 QUICK START (30 seconds):

**Option 1: Real-time Monitoring**
1. **Double-click** `Start_Real_Dashboard.bat`
2. **Open browser** to http://localhost:5000

**Option 2: Demo Mode**
1. **Double-click** `Start_Demo_Dashboard.bat`
2. **Open browser** to http://localhost:5000

**Option 3: Full Menu**
1. **Double-click** `IDS_IPS_Unified_Launcher.bat`
2. **Choose your option** from the interactive menu

## 🆘 Need Help?

- **System Health**: Run `health_check.py` in app/ folder
- **Port Issues**: Run `port_check.py` in app/ folder
- **System Debug**: Run `project_debugger.py` in app/ folder
- **Demo/Testing**: Use `Start_Demo_Dashboard.bat`

## 📁 File Guide:

- **Start_Real_Dashboard.bat** ← Real-time monitoring (recommended)

- **IDS_IPS_Unified_Launcher.bat** ← Interactive menu system
- **QUICK_START.md** ← Quick start guide
